# SortingAlgorithms
A set of sorting algorithms, written in python.

All classes utilized a function called "sort(list)".

Sorting Algorithms included so far:
  - Mergesort.py
  - InsertionSort.py
  - BubbleSort.py
  - HeapSort

Algorithms coming in the near future:
  - Quicksort
  - BucketSort
  - SelectionSort
  - RadixSort
